# Language Translator Bot

🤖 A powerful Telegram bot for translating text between 100+ languages.

## ✨ Features

- 🌍 Translate between 100+ languages
- 🔍 Auto-detect source language
- ⚡ Inline mode support
- 📱 Simple and intuitive commands
- 🆓 Completely free to use

## 🚀 Commands

- `/start` - Welcome message
- `/help` - Show help menu
- `/languages` - List all supported languages
- `/about` - About the bot
- `/translate [lang] [text]` - Translate to specific language

## 📦 Technologies

- Python 3.11+
- python-telegram-bot
- Google Translate API
- Deployed on Railway

## 🔧 Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` file with your bot token
4. Run: `python bot.py`

## 🌐 Deployment

This bot is designed for deployment on Railway:

1. Push code to GitHub
2. Connect GitHub to Railway
3. Add environment variables
4. Deploy automatically
